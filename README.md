# PayAfrik

One API. Every African Payment.

**Stripe for Africa** - Unified payment infrastructure.

## Tech Stack
- **Frontend**: Next.js 15 + TypeScript + Tailwind + ShadCN
- **Backend**: NestJS + TypeScript
- **Database**: PostgreSQL + TypeORM
- **Cache**: Redis
- **Queue**: BullMQ

## Getting Started

1. Clone the repo
2. `npm install`
3. Copy `.env.example` to `.env`
4. `npm run dev`

## Structure
- `apps/frontend/` - Merchant + Admin + Developer portals
- `apps/backend/` - NestJS API
- `packages/shared/` - Shared types & utils

Built with ❤️ for African fintech.
